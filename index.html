Window.CSS

About

To format the text and also for the patron address to fit into the window of the envelope. We wanted to print a batch of bills to send to patrons, so we are using a plugin. We attached the window.css file to the bill text so the patron address would fit in the envelope window correctly. (Envelope stock Banner reorder code 900-1001) Changed units to cm as it was easier for us.

Instructions

  1. Prepare report of Bills for printing -Use Firefox
  2. Go to Admin in Koha and then click “manage plugins”
  3. Click the “Actions Button” and then click “Run Report”
  4. Tick the patron category you wish to run the job for, and then choose Bill from the notice type.
  5. Click Submit.
  6. A table of borrower details will appear.

If the diacritics print incorrectly click view in firefox, (in print preview mode) and click text enclosing and choose Unicode. Then reload the page
